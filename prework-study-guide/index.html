HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The html tag represents the start of the document
  • The title tag defines the title for the webpage
  • The header is what is rendered on the webpage
  • The footer appears at the bottom of the page and usuall contains the author or copyright

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element.
  • Adding an asterisk will make all elements have the same style properties as the margin and padding
  • rel specifies the relationship between the document and the linked document resource
  • href specifies the location or URL of the resource
  • Declarations contain two important components being the css property we want to apply and the value of the property

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: Adds modifications to the working starting branch.
  • git commit -m "insert statement here": Creates a commit that captures a snapshot of the currently staged changes.
  • git pull origin main: Updates a branches modifications and adds them to your local computer.
  • git push origin feature/starter-code: Pushes changes made locally to the GitHub branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is a set of instructions that tells the computer how to perform a certain task. A function is different from a for loop because functions do not automatically execute when the JavaScript file is run.
  • An if statement is a way of introducing decision-making into code; the code will only execute if the statement meets a condition or is true.
  • Else if statements can be used to add more than one condition
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is realated in some way.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • A for loops contains three important statements: the first determines the starting point for our loop. The second is the condition. THe third statement is what allows the array to iterate over each item.
  • Commenting out code makes it inactive without deleting it, so that it may be edited later.
